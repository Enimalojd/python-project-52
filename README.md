## Install locally

### Requirements

- Python 3.10+
- Docker

### Steps to install

1. **Clone the repository:**

    ```sh
    git clone https://github.com/Enimalojd/python-project-52.git
    ```

2. **Create file `.env`**

3. **Run docker container:**

    ```sh
    make app
    ```
